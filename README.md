## API - Clase Persona (M√≥dulo Core)

### Constructor
```python
Persona()
```
Crea una nueva instancia vac√≠a de Persona. Utiliza el patr√≥n Builder para construcci√≥n fluida.

### M√©todos de Construcci√≥n (Builder Pattern)

#### `establecer_nombre(nombre: str) -> Persona`
Establece el nombre de la persona con validaci√≥n.
- **Par√°metro**: `nombre` (str) - Nombre de la persona
- **Retorna**: La instancia actual para encadenamiento fluido
- **Validaci√≥n**: Solo letras, acentos, espacios. Entre 2-50 caracteres
- **Excepci√≥n**: `ValueError` si la validaci√≥n falla

#### `establecer_edad(edad: int) -> Persona`
Establece la edad de la persona con validaci√≥n.
- **Par√°metro**: `edad` (int) - Edad de la persona
- **Retorna**: La instancia actual para encadenamiento fluido
- **Validaci√≥n**: N√∫mero entero entre 0 y 150
- **Excepci√≥n**: `ValueError` si la validaci√≥n falla

#### `establecer_documento_identidad(documento: str) -> Persona`
Establece el documento de identidad con validaci√≥n.
- **Par√°metro**: `documento` (str) - Documento de identidad
- **Retorna**: La instancia actual para encadenamiento fluido
- **Validaci√≥n**: Solo n√∫meros, entre 7-12 d√≠gitos
- **Excepci√≥n**: `ValueError` si la validaci√≥n falla

#### `establecer_email(email: str) -> Persona`
Establece el email de contacto con validaci√≥n.
- **Par√°metro**: `email` (str) - Direcci√≥n de email
- **Retorna**: La instancia actual para encadenamiento fluido
- **Validaci√≥n**: Formato de email v√°lido, m√°ximo 100 caracteres
- **Excepci√≥n**: `ValueError` si la validaci√≥n falla

#### `establecer_celular(celular: str) -> Persona`
Establece el n√∫mero de celular con validaci√≥n.
- **Par√°metro**: `celular` (str) - N√∫mero de celular
- **Retorna**: La instancia actual para encadenamiento fluido
- **Validaci√≥n**: Entre 8-15 d√≠gitos (acepta formatos con espacios, guiones)
- **Excepci√≥n**: `ValueError` si la validaci√≥n falla

#### `establecer_direccion(direccion: str) -> Persona`
Establece la direcci√≥n con validaci√≥n.
- **Par√°metro**: `direccion` (str) - Direcci√≥n completa
- **Retorna**: La instancia actual para encadenamiento fluido
- **Validaci√≥n**: Entre 5-200 caracteres, formato alfanum√©rico con s√≠mbolos b√°sicos
- **Excepci√≥n**: `ValueError` si la validaci√≥n falla

#### `construir() -> Persona`
Finaliza la construcci√≥n de la persona.
- **Retorna**: La instancia actual completamente construida
- **Validaci√≥n**: Verifica que al menos el nombre est√© establecido
- **Excepci√≥n**: `ValueError` si faltan datos obligatorios

### Propiedades de Acceso (Solo Lectura)

```python
@property
def nombre(self) -> Optional[str]          # Nombre de la persona
@property  
def edad(self) -> Optional[int]            # Edad de la persona
@property
def documento_identidad(self) -> Optional[str]  # Documento de identidad
@property
def email(self) -> Optional[str]           # Email de contacto
@property
def celular(self) -> Optional[str]         # N√∫mero de celular
@property
def direccion(self) -> Optional[str]       # Direcci√≥n completa
```

### M√©todos de Acceso a Datos

#### `obtener_datos_personales() -> dict`
Retorna un diccionario con los datos personales (nombre, edad, documento).

#### `obtener_datos_contacto() -> dict`
Retorna un diccionario con los datos de contacto (email, celular, direcci√≥n).

#### `obtener_todos_los_datos() -> dict`
Retorna un diccionario con todos los datos de la persona.

### Representaci√≥n de Cadena

#### `__str__() -> str`
Retorna una representaci√≥n legible de la persona con todos sus datos.

### Clase PersonaBuilder (Auxiliar)

#### `PersonaBuilder.nueva_persona() -> Persona`
Crea una nueva instancia de Persona.

#### `PersonaBuilder.con_datos_basicos(nombre: str, edad: int) -> Persona`
Crea una persona con nombre y edad establecidos.

## üìã Validaciones Implementadas

### ValidadorDatosContacto
- **`validar_email(email)`**: Valida formato de email est√°ndar
- **`validar_celular(celular)`**: Valida n√∫mero de celular de 8-15 d√≠gitos
- **`validar_direccion(direccion)`**: Valida direcci√≥n de 5-200 caracteres
